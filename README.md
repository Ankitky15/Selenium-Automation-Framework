# Selenium Automation Framework

A Selenium-based test automation framework developed using Java, Selenium WebDriver, TestNG, and Maven. The framework follows the Page Object Model (POM) design pattern to automate web application testing and improve test execution efficiency, maintainability, and reusability.

---

## Features

- Automated UI testing for web applications
- Page Object Model (POM) architecture
- TestNG for test execution and assertions
- Maven for dependency management
- Extent Reports for detailed test reports
- Screenshot capture on test failure
- Cross-browser testing support
- Reusable utility methods
- Easy configuration using properties file

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/Selenium-Automation-Framework.git
```

2. Navigate to the project directory

```bash
cd Selenium-Automation-Framework
```

3. Install dependencies

```bash
mvn clean install
```

4. Execute the test suite

```bash
mvn test
```

---

## Reports

After execution, Extent Reports are generated inside the `test-output` directory.
